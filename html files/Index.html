Project Summary

The project analyzed how weather changes as we approach the equator. Data was complied from the OpenWeatherMap API and utilizing Jupyter notebook. The analysis was performed using approximately 500 cities throughout the world.

Once the weather data for each city was obtained Matplotlib was utilized to plot various aspects of the weather against the latitude of the locations. The factors that were analyzed in the project include temperature, cloudiness, humidity, and wind speeds. This site contains the source data along with visualizations that were created during the project. Also included are explainations and descriptions of any trends and correlations observed.
